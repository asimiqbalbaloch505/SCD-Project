# Quiz Master 🎯

A simple Java Swing-based Quiz Application that allows users to test their general knowledge with multiple-choice questions. This project is great for learning GUI development in Java using Swing.

---

## 🧠 Features

- Welcome screen with user input
- 10 multiple-choice questions
- Score calculation
- Result screen showing score and exit option
- Clean, user-friendly interface
